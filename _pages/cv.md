**Geographical concentration index (R package)**
 * <font size="-0.5">In construction.</font>

<p style="text-align: justify"> <font size="-1"> <strong> Description: </strong> It contains land-unit based and distance based concentration indices. In particular, land-unit-based indices allow to measure concentration within and between urban systems while distance-based indices inform of the co-agglomeration. The code also constructs confidence intervals based on simulations of a random process repeated one thousand times. It contains the Theil Index and decomposes it on a within and between measure. </font>
</p>
